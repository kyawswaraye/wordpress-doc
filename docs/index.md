## CMS(Content Management System)

CMS ဆိုသည်မှာ Content Management System ဖြစ်ပြီး Website Data များကိုအလွယ်တကူစီမံ ခန့်ခွဲနိုင်သော panel တစ်ခုဖြစ်ပါတယ်။

## What is Wordpress

**Wordpress**
wordpress မှာ စွမ်းဆောင်ရည်များစွာလုပ်ဆောင်နိုင်ဆောင်နိုင်ပြီး ကမ္ဘာပေါ်ရှိ အကြီးမားဆုံး website တစ်သန်းခန့် (၁၄ရာခိုင်နှုန်းသော ကမ္ဘာပေါ်ရှိ website) များဟာ wordpress ကိုအသုံးပြုထားကြသည်။wordpress ဆိုတာ နားလည်ရလွယ်ပြီး ခက်ခဲတဲ့ နည်းပညာတွေ တတ်မြောက်မထားပဲ လှပသပ်ရပ်တဲ့ website တွေကို အချိန်တိုလေးနဲ့ ဖန်တီးနိုင်ပါတယ်။
website တစ်ခုကို custom code နဲ့ ရေးမယ်ဆိုရင် အနည်းဆုံး အချိန်တစ်လလောက်ယူရလေ့ ရှိပေမယ့် wordpress ကိုသာ သုံးလိုက်မယ်ဆိုရင် ရက်ပိုင်းလေးနဲ့ ပြီးမြောက်သွားနိုင်ပါတယ်။

wordpress ၂ မျိုးရှိပါတယ်။wordpress.com နဲ့ wordpress.org ဖြစ်ပါတယ်။

wordpress.com ဆိုသည်မှာ wordpress အဖွဲ့မှ commercial အဖြစ်ရောင်းစားသော CMS တစ်ခုဖြစ်ပါတယ်။

wordpress.org ဆိုသည်မှာ open source ပေးထားသော CMS တစ်ခုဖြစ်ပြီး ငွေကြေးကုန်ကျခြင်းမရှိပဲ မိမိစိတ်ကြိုက် လိုအပ်သလိုရေးဆွဲနိုင်ပါတယ်။ဒါပေမယ့် wordpress.com လိုမျိုးမဟုတ်ပဲ ကိုယ်တိုင်အစအဆုံး ရေးယူရသော CMS တစ်ခုဖြစ်ပါတယ်။
